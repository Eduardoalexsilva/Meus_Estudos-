Minha Página

Neste exemplo, a barra lateral está oculta (style = "display: none")

Ele é mostrado quando você clica no ícone do menu no canto superior esquerdo.

Quando é aberto, ele desloca o conteúdo da página para a direita.

Usamos JavaScript para adicionar uma margem esquerda de 25% ao elemento div com id = "main" quando isso acontece. O valor "25%" corresponde à largura da barra lateral.
